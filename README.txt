Antigrief Mod v1.0
By Leslie Krause

Repository
----------------------

  * Browse source code:
    https://bitbucket.org/sorcerykid/antigrief

  * Download archive:
    https://bitbucket.org/sorcerykid/antigrief/get/master.zip
    https://bitbucket.org/sorcerykid/antigrief/get/master.tar.gz

Revision History
----------------------

Version 1.0b (09-Apr-2019)
  - initial beta version

Dependencies
----------------------

None

Compatability
----------------------

Minetest 0.4.14+ required

Installation
----------------------

  1) Unzip the archive into the mods directory of your game
  2) Rename the antigrief-master directory to "antigrief"
  3) Add "antigrief" as a dependancy of the Default mod
